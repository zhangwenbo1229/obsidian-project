# Obsidian Project 手工验收清单

## 安装与生命周期

- [x] 全新 Vault 启用后生成 `项目管理/全局配置.md`。
- [x] 禁用并重新启用插件后没有重复监听、视图或通知。
- [x] `npm run build` 后三个产物出现在测试 Vault 插件目录。
- [x] 热重载后开发者控制台没有异常。

## 项目与配置

- [x] 添加人员并切换当前用户后，重载仍保留配置。
- [x] 创建项目后生成单独项目配置 Markdown。
- [x] 任务类型模板、自定义字段和工作流修改后可重新解析。
- [x] 修改自定义字段键后，项目内所有任务同步更新。
- [x] 项目存在任务时禁止删除。
- [x] 修改项目代码后，任务 Key、文件名、目录和关系链接同步更新。

## 任务

- [x] 连续创建两个任务，Key 单调递增。
- [x] 删除任务后下一编号不复用。
- [x] 按月分组项目把任务放入正确 `YYYY-MM` 目录。
- [x] 类型模板只进入 **任务正文** 区段。
- [x] 必填自定义字段为空时禁止创建或保存。
- [x] 直接增加未知 frontmatter 后，通过弹窗保存仍保留该字段。
- [x] 修改 UUID 或 Key 时，运行中的插件恢复最近有效值。

## 工作流、关系与备注

- [x] 非法看板拖拽回弹并提示。
- [x] 首次进入处理中自动填写开始日期。
- [x] 正常完成与终止分别维护完成日期和终止日期。
- [x] 重新打开清空两个结束日期。
- [x] 禁止自引用、循环父子关系和跨项目父子关系。
- [x] 普通关联在双方任务详情中可见。
- [x] 备注可以新增、修改、删除，并能直接编辑 Markdown。

## 迁移与删除

- [x] 无父子关系任务可迁移并获得目标项目新 Key。
- [x] 单独迁移有父子关系的任务被阻止。
- [x] 迁移整棵任务树后父子关系保持有效。
- [x] 同名同类型自定义字段自动映射。
- [x] 删除有父任务或子任务的任务被阻止。
- [x] 删除普通关联任务时清理其他文件的结构化关系。

## 视图与移动端

- [x] 个人视图统计口径符合开发规格。
- [x] 项目筛选不同条件使用 AND，同条件多值使用 OR。
- [x] 列表临时列、排序和分批渲染正常。
- [x] 看板合法拖拽更新状态。
- [x] 日历不可拖动修改日期。
- [x] 移动端可以查看和编辑，但没有新增任务入口。

## 同步与异常

- [x] 制造重复 Key 后，两份任务都退出统计并显示修复入口。
- [x] 制造重复 UUID 后，用户可以为选中副本生成新 UUID。
- [x] 删除关系目标文件后显示缺失关系并可清理。
- [x] 删除固定正文标题后可重建结构且不覆盖正文。
- [x] 中断批量迁移后，`项目管理/.迁移/` 记录每个任务完成状态。

# obsidian-project

`obsidian-project` 是一个本地优先的 Obsidian 项目管理与个人工作台插件。所有数据均保存在 Vault 内的 Markdown 文件中，可参与 Obsidian Sync、Git 或其他文件同步。

## 功能

### 项目管理
- Jira 风格的项目代码和不可复用任务编号，例如 `PROJ-123.md`。
- UUID v4 永久标识，任务迁移项目后仍保持身份不变。
- 项目级任务类型、正文模板、自定义字段和可配置工作流。
- 项目可配置图标和颜色，在三个视图中统一展示。
- 项目内父子任务、关联关系和可编辑备注。
- 结构化子任务保存在父任务 Markdown 中，支持独立编辑弹窗。
- 项目代码、任务项目、自定义字段键和完整任务树迁移。
- 可恢复的持久化迁移日志；中断后可从数据问题中心继续。

### 个人工作台
- 可拖拽、缩放、排序和独立配置的仪表盘，支持 17 种卡片类型：
  - **任务卡片** - 项目/任务统计（数量、百分比、进度条、半圆仪表）
  - **天气卡片** - 支持 Open-Meteo、和风天气、OpenWeatherMap
  - **日历卡片** - 农历、天干地支、传统节日、24 节气、签到标记
  - **公网 IP 卡片** - 显示公网 IP 和中文地理位置（含运营商）
  - **笔记统计卡片** - 笔记数、字符数、文件夹数、总大小，支持元数据筛选和字段类型选择
  - **待办卡片** - 卡片内直接完成，支持行内编辑
  - **最近文件卡片** - 最近创建、最近编辑、常用文件
  - **新闻卡片** - RSS/Atom 订阅，支持分页
  - **目录卡片** - 目录树浏览
  - **文本卡片** - Markdown 实时预览编辑
  - **图表卡片** - 柱状图、折线图、饼图，支持 CSV 导入
  - **日期卡片** - 当前日期时间显示
  - **倒计时卡片** - 自定义事件倒计时
  - **签到卡片** - 每日签到追踪
  - **时间进度卡片** - 日/周/月/年进度
  - **热力图卡片** - 文件变更热力图
  - **计算器卡片** - 内置计算器
  - **网页卡片** - iframe 嵌入外部网页
- 每张卡片可设置背景颜色；右键复制卡片配置。
- **设置 → 个人仪表盘** 可控制允许新增的卡片类型。

### 任务视图
- 专用任务视图，支持按项目分组、搜索、筛选和排序。
- 左侧边栏按项目导航，显示项目图标和颜色。
- 任务行内编辑元数据（优先级、经办人、日期等）。
- 已完成任务显示/隐藏切换。

### 人员管理
- 全局人员列表，支持名称图标和颜色配置。
- 人员元数据图标使用项目元数据选择器，统一 UX。
- 人员类型字段（报告人、经办人）统一使用人员名称属性控制图标和颜色。

### 项目视图
- 列表、看板、日历、四象限四种视图模式。
- 可配置筛选器（状态、类型、人员、标签、日期等），支持保存和恢复。
- 列宽拖拽调整、排序、搜索。

### 数据安全
- 所有数据以 Markdown 为主存储，`data.json` 只保存 UI 状态。
- **设置 → 配置数据** 支持导入/导出，排除 API 密钥。
- 重复 UUID、重复 Key、失效关系检测。
- 移动端完整支持查看和编辑。

## 安装

将以下发布文件复制到 `<Vault>/.obsidian/plugins/obsidian-project/`：

- `main.js`
- `manifest.json`
- `styles.css`

随后在 **设置 → 第三方插件** 中启用 **obsidian-project**。

## 首次使用

1. 打开 **设置 → obsidian-project**。
2. 确认全局配置路径，默认是 `项目管理/全局配置.md`。
3. 设置当前用户并维护共享人员列表。
4. 创建项目，配置任务目录、任务类型、模板、自定义字段和工作流。
5. 执行命令 **obsidian-project: 新增任务**。

## 本地开发

要求 Node.js 20+ 和 npm。

```powershell
npm install
$env:OBSIDIAN_VAULT_PATH='D:\path\to\vault'
npm run dev
```

每次 watch 编译成功后会自动复制发布产物到测试 Vault。生产构建：

```powershell
npm test
npm run lint
npm run build
```

## 隐私与安全

- 插件不包含遥测，也不执行远程代码；所有联网卡片默认关闭，需要逐卡片明确启用。
- 天气卡片启用后只向用户选择的 Open-Meteo、和风天气或 OpenWeatherMap 发送经纬度。
- 公网 IP 卡片访问 `api.ip.sb` 和 `ip-api.com` 获取 IP 和中文地理位置。
- 新闻卡片只访问用户填写的 RSS/Atom 地址，以纯文本渲染。
- 网页卡片在带 `sandbox` 的 iframe 中打开，只应嵌入可信站点。
- 插件只通过 Obsidian API 读写当前 Vault。
- 新闻与自定义天气主机仅允许公共 HTTPS 地址，拒绝本机、回环、私网地址。

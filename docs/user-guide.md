# Obsidian Project 用户指南

## 个人视图

在个人视图空白处右键可新增自定义卡片。每张卡片可独立设置背景色、内容范围与显示参数；右下角保留不可见的拖动热区用于调整大小。

**设置 → 个人视图** 单独管理可新增的卡片类型。天气服务的 API 密钥和和风天气接口主机也在此统一配置，天气卡片仅保留数据源、位置、预报天数和刷新策略。天气和资讯卡片只有在单卡开启联网后才会请求外部服务，不会上传库、笔记或任务内容。

- 日期卡片可显示当前日期、星期、实时钟、农历和固定节假日。
- 日历卡片可分别开关农历和节假日。
- 待办卡片收集指定目录 Markdown 文件中的未完成复选框任务，并支持排除目录。
- 图表卡片的折线图和柱状图可开关坐标轴、图例、数据标签并配置颜色。
- 倒计日按本地自然日计算与目标日期的天数差。
- 热力图统计整个库中各文件当前的最后修改日期。Obsidian 不提供历史编辑次数，因此它不表示历史上每天发生的全部修改次数。
- 所有卡片均可通过右键菜单复制，副本会保留原卡片数据源、尺寸和显示配置。
- 日历会显示农历年月日、天干地支年、公历节日、传统农历节日和 24 节气；今日使用主题色高亮，周末日期使用特殊颜色。
- 待办卡片的复选框会直接修改来源 Markdown 行。若笔记已经变化导致原行不再是任务，插件会拒绝覆盖。
- 百分比卡片可使用任务统计或手工输入的当前值/目标值，并选择百分比数字或进度条。
- 笔记统计可按文件后缀、元数据属性及其值筛选，并控制统计字段及顺序。
- 文件卡片支持最近创建、最近编辑和常用文件。常用文件按照本次升级后插件记录的文件打开次数排序，无法追溯此前历史。

## 数据目录

默认目录如下：

```text
项目管理/
├─ 全局配置.md
├─ 项目配置/
└─ 任务/
```

项目可以使用自己的任务目录，并选择按 `YYYY-MM` 创建月份分组。插件会索引 Vault 中所有带 `pm-kind: task` 的 Markdown，因此手动移动任务文件不会让任务失效。

## 人员与项目

在 **设置 → obsidian-project** 中选择当前用户、添加共享人员，并创建或编辑项目。

项目配置支持：

- 项目名称、启用状态和任务目录
- 任务类型、图标、颜色及正文模板
- 十种自定义字段类型
- 待处理、处理中、正常完成和终止未完成状态
- 明确的工作流转换
- 项目代码迁移和项目删除检查

> [!warning]
> 已有任务的项目代码只能通过配置界面的迁移操作修改。请勿直接编辑任务 Key。

## 创建任务

在命令面板执行 **obsidian-project: 新增任务**，填写项目、类型、标题、经办人、日期、标签、自定义字段和正文。插件先持久化项目计数器，再创建任务；创建失败也不会复用已经分配的编号。

移动端不提供新增任务命令，但可以查看和编辑已有任务。

## 编辑任务

在个人视图或项目视图选择任务，打开统一编辑弹窗。弹窗支持：

- 修改标题、经办人、日期、正文和自定义字段
- 执行合法的工作流转换
- 添加一个父任务或多个普通关联
- 新增、修改和删除备注
- 迁移当前任务或整棵任务树
- 在 Markdown 中打开
- 经过二次确认后移入 Obsidian 回收站

任务的 `uid` 和 `key` 在弹窗中只读。插件运行期间若检测到直接修改，会恢复最近有效值；重新启动或同步后无法确定原值时会进入数据问题中心。

## 视图

### 个人视图

个人视图是可自由编排的卡片工作台。拖拽卡片可调整位置，拖动右下角可调整大小；在空白处右键可新增卡片，在卡片上右键可打开该卡片的独立设置或删除自定义卡片。

可添加的卡片包括：

- 数字、百分比和任务列表：可绑定保存的项目筛选器；任务列表可单独配置字段与排列方向。
- 天气：配置地点名称、经纬度、预报天数和数据源，明确允许联网后显示当前天气和近期预报。Open-Meteo 免 Key；和风天气需要 API Key 与控制台分配的 HTTPS API Host；OpenWeatherMap 需要 API Key。
- 日历：支持切换月份、星期起始日、固定节日和可选农历。
- 笔记统计：按目录统计 Markdown 笔记数、字符数、目录数和笔记分布。
- 最近文件：按修改时间显示并打开指定目录内的 Markdown 文件。
- 资讯：添加 RSS/Atom 地址并明确允许联网后，以纯文本分页显示订阅内容。
- 目录：按指定根目录和最大层级浏览、展开并打开 Markdown 文件。
- 文本：输入 Markdown 内容，并在卡片设置中实时预览 Obsidian Markdown 渲染结果。
- 图表：支持折线图、柱状图和饼图；可直接编辑数据表格、增删行与系列，也可粘贴 CSV。折线图和柱状图支持多系列，饼图使用第一组数据。

天气与资讯卡片默认不联网；每张卡片都需要在自己的设置中单独启用。

和风天气和 OpenWeatherMap 的 API Key 会随个人视图卡片配置保存在插件 `data.json` 中。请不要在共享或公开的配置文件中填写私人密钥。

天气卡片可配置 1–7 天的预报范围。笔记统计和最近文件卡片支持按 Vault 相对路径排除目录及其全部子目录。所有卡片都可独立设置背景颜色，颜色会与当前 Obsidian 主题背景混合以保持可读性。

在 **设置 → obsidian-project → 视图显示 → 个人视图自定义卡片** 中可以控制右键新增菜单显示哪些卡片类型。关闭类型不会删除已经创建的卡片。

### 项目视图

先选择项目，再使用关键词、状态、状态分类、类型、人员、标签、日期和自定义选项组合筛选。

- **列表**：临时选择列、按列排序，并分批渲染大量任务。
- **看板**：按状态分列，拖拽只允许执行工作流中存在的转换。
- **日历**：按开始日期和计划完成日期只读展示。

## 数据问题

个人视图的 **数据问题** 按钮列出解析错误、孤立任务、重复 UUID、重复 Key、Key 与文件名不一致、缺失关系和正文结构错误。

可自动处理的问题包括：

- 按 Key 修正文件名
- 为重复 Key 重新编号
- 为选中的 UUID 冲突副本生成新 UUID
- 清理缺失关系
- 重建固定正文区段

其他问题可以选择 **打开文件** 后手动修复。未完成迁移会显示已完成文件数和总文件数；选择 **继续迁移** 后，插件从日志中的新旧路径与任务快照安全恢复。

## 多设备同步

两台离线设备可能同时生成相同 Key。同步后插件会暂停冲突任务参与统计，并要求用户选择一个文件重新编号，不会静默重命名。迁移日志保存在 `项目管理/.迁移/`，记录旧/新 Key、旧/新路径和逐文件完成状态，可在中断后继续。

## Markdown 结构

任务必须包含以下固定区段：

```markdown
## 任务正文

## 链接

## 备注
```

结构化关系和备注使用不可见 HTML 注释保存 UUID。普通 Wikilink 和未知元数据会被保留，不参与结构化查询。
